# Presentation

## How to Use

### Show the Slides

Git clone this repo, and use Web browse to open the HTML file of each slide under `slides` folder.

For example, if git clone this repo into `/Users/robin/gocode/src/github.com/supereagle/presentation`, then go to the 
sub folder `slides/3w-of-devops` and right click the HTML file `3w-of-devops.html`, select the web browse as the open 
method like `Google Chrome`. You can also directly input the file path 
`file:///Users/robin/gocode/src/github.com/supereagle/presentation/slides/3w-of-devops/3w-of-devops.html` in your 
web browse to open it.

### PDF Export

1. Open your presentation with print-pdf included in the query string, i.e. 
`file:///Users/robin/gocode/src/github.com/supereagle/presentation/slides/3w-of-devops/3w-of-devops.html?print-pdf`.
2. Open the in-browser print dialog (CTRL/CMD+P).
3. Change the **Destination** setting to **Save as PDF**.
4. Change the **Layout** to **Landscape**.
5. Change the **Margins** to **None**.
6. Enable the **Background graphics** option.
7. Click **Save**.
